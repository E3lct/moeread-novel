# 墨读 Moeread 项目记忆

## 项目概况
- 大二 Java Web 期末作业，小说阅读器
- 技术栈：JSP + Servlet + JavaBean + JDBC + DAO + MVC, MySQL 8, Tomcat 9
- 不用框架，原生写，怕老师看出非本人所写
- GitHub 用户名：E3lct，仓库名：moeread-novel（待创建，gh CLI 已装待登录）

## 作业版功能清单（7个页面）
1. 首页：今日阅读环(18/30min，左上角满琥珀黄背景白字) + 推荐位小组件(右侧，封面+简介) + 最近在读 + 喜爱书籍（圆形进度环，悬停显示）
2. 书架：网格/分组视图切换 + 系统标签(喜欢/准备看/已看完/正在看) + 自定义标签(可新建/编辑) + 搜索 + 编辑书名封面 + 点击弹书籍详情弹窗(开始阅读/续读)
3. 阅读器：章节目录侧栏(可收起) + 字号/行距/背景色/夜间模式 + 键盘翻页 + 自动保存进度 + ESC摸鱼
4. 统计：年度热力图(颜色深浅=阅读时长) + 总时长/书籍数/连续打卡(3卡片) + 已读完书籍列表(悬停显示阅读时长)
5. 设置：个性化(导入看板娘图片+透明度滑块) + 阅读目标(默认30min/天，+-按钮调节) + 导入历史(成功/失败状态) + 账号(修改密码/退出) + 关于
6. 导入：txt单文件 + ZIP批量，自动解析章节
7. 摸鱼模式：VS Code伪装界面(深色/浅色可切)，阅读内容藏在代码注释里，活动栏可收起

## 不做的功能
在线书源拉取、PDF/EPUB、轻小说对话排版、阅读笔记、PWA/APP封装、听歌歌词

## 设计规范
- 风格：琥珀黄温暖配色，参考阅读器护眼色调 + Figma Amber 色板
- 主色：#F59E0B 琥珀黄，浅色 #FBBF24 / #FEF3C7 / #FFFBEB，深色 #D97706 / #B45309 / #92400E
- 背景：#FAF8F3 暖米白
- 卡片：#FFFFFF 纯白 + 柔和阴影 rgba(217,119,6,0.06~0.18)
- 主文字：#3D2E1A 深棕
- 次要文字：#78350F / #92400E
- 提示文字：#A16207
- 边框/分割线：#FDE68A 浅琥珀
- 今日阅读组件：底部满琥珀黄渐变背景，白字印在上面（左上角占位）
- 推荐位组件：浅琥珀背景 + 书籍封面 + 简介（今日阅读右侧）
- 书籍封面：琥珀黄系渐变（深浅/方向交替），白字书名，柔和阴影
- 底部悬浮导航栏：胶囊形白底+琥珀阴影，4个文字标签（首页/书架/统计/设置），选中态琥珀渐变白字
- 圆形进度环：白色描边（在琥珀背景上）/ 琥珀描边（在白底上），悬停时显示
- 看板娘：右下角 fixed 定位，设置页可导入图片，可调节透明度
- 绝对不用火焰/导航等图标，只用文字标签。唯一允许的图标是爱心（琥珀色#D97706）
- 前端可用 Vue（老师允许）

## 用户偏好（硬性规则）
- 用户上课没认真听，需要解释技术概念
- 时间不是问题，做好就行
- 担心UI不好看，要先审核设计稿再写代码
- 每完成大版本要 commit 到 GitHub
- **绝对禁止在项目中使用表情符号(emoji)！** 用户非常反感，看起来很突兀。所有图标用 SVG/CSS 实现。
- 书籍卡片设计：封面占满整个框，底部玻璃质感毛玻璃条显示书名，圆形进度环默认隐藏，仅鼠标悬停时显示
